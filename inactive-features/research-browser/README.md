# Research Browser (inactive archive)

Status: inactive and unsupported as of 2026-07-18.

This directory preserves the former embedded Research Browser implementation,
its security policy, tests, verifier scripts, and historical documentation for
future reference. Nothing here is imported, typechecked, tested, bundled, or
packaged by the active Venice Forge application.

The supported Research workspace remains active. It provides provider search,
Venice/Jina scraping, AI research, citations, saved sessions, document upload,
findings, summaries, and workflow/prompt-library handoffs without an embedded
browser surface.

Reactivation requires a new reviewed work order, a fresh threat model, current
Electron navigation containment tests, and explicit restoration of build and
CI coverage. Do not import files directly from this archive.
